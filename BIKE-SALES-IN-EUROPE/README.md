# Bike Sales Analysis - Europe 🇪🇺

**End-to-End Excel Data Analysis & Interactive Dashboard**

### 📋 Project Overview
This project analyzes a **Bike Buyers** dataset to uncover key factors influencing bike purchases. It includes full data cleaning, feature engineering, pivot table analysis, and an interactive dashboard.

### 🎯 Business Objective
Help the sales & marketing team identify high-potential customer segments based on demographics, income, and behavior.

### 🛠️ Tools Used
- **Microsoft Excel** (Advanced)
  - Data Cleaning & Transformation
  - Formulas (IF, etc.)
  - Pivot Tables & Slicers
  - Interactive Dashboard

### 🔧 What I Did

- Cleaned the dataset (removed duplicates, standardized data)
- Created **"Age Bracket"** column for better analysis  
  (Gen Z: 25-34 | Gen X: 35-50 | Old: >50)
- Built 3 Pivot Tables:
  1. Average Income by Gender (Purchased vs Not)
  2. Purchase Count by Commute Distance
  3. Purchase Count by Age Bracket
- Designed a clean **Interactive Dashboard** with multiple slicers

### 📊 Key Insights
- Customers living **0-1 miles** from work have the highest purchase rate.
- **Gen X (35-50)** shows the strongest buying behavior.
- Male buyers tend to have higher average income.
- Clear relationship between commute distance, age, and purchase decision.
